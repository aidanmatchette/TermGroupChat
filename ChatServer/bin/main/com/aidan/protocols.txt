User & Server

1. User --> Server

    * login / logoff
    * status

2. Server --> User

    * online / offline messages

3. User --> User

    * direct messages
    * broadcast messages

Commands:

    login <user> <password>
    logoff

    msg <usr> body.. 
    guest:-> "msg jim Hello World"
    jim:-> "guest Hello World"


#topic <--- chat room / group chat for certain topic
join #<topic name>
send: msg #topic body...
receive: msg #topic:<login> body...
leave #<topic name>
